Scanner Help & Guide
Quick Start
  1. Point the camera at the student's QR code
  2. Wait for the beep and scan result
  3. Check the notification for scan status
  4. Use the IN/OUT toggle to switch scan modes
Scan Modes
IN Mode (Green)
Records student entry to campus or event
OUT Mode (Red)
Records student exit from campus or event
Offline Mode

When internet is unavailable, scans are saved locally and will sync automatically when connection is restored.

What to know:
  • Yellow banner shows offline status
  • Scans are saved to device (max 500)
  • Same student can't be scanned twice (duplicate prevention)
  • Sync happens automatically when online
Troubleshooting
QR code won't scan?
• Ensure good lighting
• Hold QR code steady and flat
• Try manual entry (registered name or student ID)
• Check if camera permission is granted
Scanner not working?
• Refresh the page (Ctrl+Shift+R)
• Try switching camera (if multiple available)
• Check browser camera permissions
• Use manual entry as backup
Duplicate scan warning?
• Student already scanned (same mode)
• Wait 3 seconds before re-scanning
• Toggle IN/OUT if needed
• Check recent entries list
Keyboard Shortcuts
Ctrl + I Switch to IN mode
Ctrl + O Switch to OUT mode
Ctrl + M Focus manual entry field
Esc Close popups/modals
Pro Tips
  • Keep the scanner page open during your shift
  • Check "Live Entries" to see recent scans
  • Use event selector for event attendance tracking
  • Manual entry: check out with name/ID only (blank reason), Enter or Submit; check in requires a reason
  • Offline queue syncs automatically - no action needed
Need more help? Contact your supervisor or IT support.
