Import students from CSV (Registrar)

Upload a CSV file. Student ID is optional — leave the column out or cells blank and the system assigns a unique 8-digit ID for each new row.

Full format (with header): optional student_id, then first_name, middle_name (Middle Initial), last_name, email, address, birthdate, required sex (or gender; MALE or FEMALE, or M/F), contact_number, guardians_parents, guardian_contact, program or course (same column), year_level, section. Birthdate: MM/DD/YYYY or YYYY-MM-DD.

Simple format (with header): first_name, last_name, optional email, and required sex (or gender). You can still include student_id if you already have official IDs.

QR codes are generated automatically — view them in the student list or on each student’s edit page.
